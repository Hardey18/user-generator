# User Generator

Click [here](https://usergenerator.netlify.app/) to the deployed site

## Technologies used in this project include
- React
- JavaScript
- Styled Components
- React Icons
- Playwright
- React JS Pagination
- React Switch

## Features
* Pagination
* Search field by firstname
* Search field by country name
* View each user
* CSV Download
